STYLE RULES:

1. THE ANSWER IS THE PAYOFF. The blank should hide the surprising, unexpected, or funny word — the one that makes you go "wait, really?" Once revealed, it should feel satisfying.

2. INCLUDE CONTEXT. Don't just state the dry fact. Add a parenthetical, a clause, or extra detail that makes the question more interesting even while guessing. The question itself should be fun to read.
   BAD:  Mantis shrimp can punch very fast.|Shrimp
   GOOD: Mantis \_\_\_\_\_ can punch with the speed of a .22 caliber bullet, generating enough heat to boil the surrounding water instantly.|Shrimp

3. some examples of potential subject areas of questions for inspiration: Animal/biology weirdness,Food/drink,Geography/records,Historical weirdness,Human body,Language/words,Pop culture/tech,Science/physics 


4. ANSWERS should be:
   - Usually a single word (a noun, verb, animal, person, place, brand, number)
   - Occasionally a short phrase (2–3 words max like "Sea otters", "Black Death", "Mandela Effect")
   - Never a full sentence
   - Case-insensitive at guess time, but store with natural capitalization

5. UNDERSCORE COUNT must exactly match the character count of the answer. Count every letter in the answer (including spaces for multi-word answers... wait, actually: use ONE backslash-underscore pair per CHARACTER, and put them back-to-back with no gaps). For multi-word answers, the underscores should still all be run together (the display code handles word spacing automatically).
   - "Owls" = 4 chars = \_\_\_\_ (four \_ pairs)
   - "Sea otters" = 9 chars not counting the space = \_\_\_\_\_\_\_\_\_ (nine \_ pairs) — just count all non-space chars... actually count ALL chars including the space: "Sea otters" = 10 chars = \_\_\_\_\_\_\_\_\_\_ (ten \_ pairs)

6. DIFFICULTY should be medium. Not so easy that everyone knows it immediately, not so obscure that nobody has a chance. The sweet spot is "I should have known that" or "oh wow I never knew that."

7. AVOID:
   - Questions that are essentially the same as an existing one even if different words
   - Purely academic facts with no interesting angle (e.g., "Water is made of two \_\_\_\_\_\_\_ atoms.|hydrogen")
   - Questions where the blank could have multiple valid-sounding answers
   - Answers that are too long (4+ words)
   - Topics that were already covered with a better version in the existing set
